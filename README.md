## 🔧 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/kanat-portfolio.git
   cd kanat-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Open [http://localhost:3000](http://localhost:3000)**
